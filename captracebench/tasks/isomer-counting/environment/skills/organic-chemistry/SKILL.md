# Organic Chemistry: Isomer Counting

## Overview

This skill covers counting structural (constitutional) isomers for alkane molecules given their molecular formula.

## Alkane Formulas

Alkanes follow the formula **CnH2n+2** (saturated hydrocarbons).

## Counting Structural Isomers

Structural isomers have the same formula but different connectivity. For alkanes:

### Rules for Valid Alkane Structures

1. Each carbon has exactly 4 bonds
2. No double bonds or rings (saturated)
3. All carbons must be connected in a chain or branched arrangement

### Method

For a given formula CnH2n+2:
1. Determine if straight-chain (no branching) is possible → 1 isomer
2. Count branched possibilities:
   - 2-methyl branch (isobutane) for C4
   - 2-methyl and 3-methyl branches for C5+
   - More branches for larger molecules

### Known Isomer Counts

| Formula | # Isomers | Names |
|---------|-----------|-------|
| C1H4 | 1 | methane |
| C2H6 | 1 | ethane |
| C3H8 | 1 | propane |
| C4H10 | 2 | butane, isobutane |
| C5H12 | 3 | pentane, isopentane, neopentane |
| C6H14 | 5 | hexane isomers |
| C7H16 | 9 | heptane isomers |
| C8H18 | 18 | octane isomers |

### Validation

To verify your count:
- Each isomer must have unique connectivity
- Branching patterns that are identical by symmetry count as one
- Mirror images (enantiomers) do not count for structural isomers

## Example

**C4H10 (butane)**: 2 isomers
- n-butane: CH3-CH2-CH2-CH3 (straight chain)
- isobutane: CH3-CH(CH3)-CH3 (branched)

**C5H12 (pentane)**: 3 isomers
- n-pentane: CH3-CH2-CH2-CH2-CH3
- isopentane (2-methylbutane): CH3-CH(CH3)-CH2-CH3
- neopentane (2,2-dimethylpropane): C(CH3)4
