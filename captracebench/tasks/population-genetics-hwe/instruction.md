# Hardy-Weinberg Equilibrium Task

Given allele counts for a biallelic locus in `/root/input.txt`:
- Number of homozygous reference (AA): A
- Number of heterozygous (Aa): H
- Number of homozygous alternate (aa): a

Calculate the expected genotype counts under Hardy-Weinberg equilibrium and the expected heterozygosity.

Write results to `/root/output.txt`:
```
Expected AA: X
Expected Aa: X
Expected aa: X
Expected heterozygosity: X.XXXX
Observed heterozygosity: X.XXXX
```

Round to 4 decimal places.
