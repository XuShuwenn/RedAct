# Constrained Optimization Task

Minimize the function f(x) = x1² + x2² subject to the constraint x1 + x2 ≥ 1.

Use pymoo with NSGA-II to find the optimal solution with bounds 0 ≤ x1, x2 ≤ 10.

Write to `/root/output.txt`:
```
Optimal x1: X.XX
Optimal x2: X.XX
Objective: X.XX
```

Round to 2 decimal places.
