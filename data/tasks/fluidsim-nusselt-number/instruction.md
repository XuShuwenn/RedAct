# Nusselt Number Calculation Task

Given a forced convection heat transfer scenario with the following parameters in `/root/input.txt`:
- h: convective heat transfer coefficient (W/m²·K)
- k: thermal conductivity (W/m·K)
- D: characteristic length (m)

Calculate the Nusselt number: Nu = h × D / k

Write result to `/root/output.txt`:
```
Nusselt number: X.XX
```

Round to 2 decimal places.
