# Task: Count Structural Isomers

Given a molecular formula, determine the number of possible structural (constitutional) isomers.

## Input

The molecular formula is provided in `/root/input.txt` (e.g., C4H10).

## Output

Write your answer to `/root/output.txt`. Format:
```
Number of isomers: [count]
```

Consider only structural (constitutional) isomers, not stereoisomers.

## Example

Input: C4H10 → Output: `Number of isomers: 2` (butane and isobutane)

## Your Task

Read the molecular formula from `/root/input.txt` and write your answer to `/root/output.txt`.
